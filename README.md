# YM-бот — Яндекс.Музыка в голосовом чате

Воспроизведение Яндекс.Музыки в голосовых каналах lolka.app. Кнопочное управление, бесконечное радио, поиск.

## Быстрый старт

1. Установи **Node.js** — [nodejs.org](https://nodejs.org)
2. Установи **ffmpeg** — `winget install "FFmpeg (Essentials Build)"`
3. Получи **токен бота** — создай бота на [lolka.app](https://lolka.app), скопируй токен
4. Открой `start.bat` двойным кликом

Всё. start.bat сам:
- Установит зависимости
- Запустит визард настройки (попросит токен бота и проведёт авторизацию в Яндексе)
- Запустит бота

## Команды

| Команда | Описание |
|---------|----------|
| `!ym <текст>` | Поиск треков |
| `!ym 1-5` | Выбрать трек из поиска |
| `!ym wave` | Моя волна |
| `!ym radio <жанр>` | Радио по жанру |
| `!ym pl` | Плейлисты |
| `!ym top` | Чарт |
| `!ym new` | Новинки |
| `!skip` / `!stop` / `!loop` | Управление |
