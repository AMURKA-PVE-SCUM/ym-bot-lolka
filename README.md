## Быстрый старт

### Требования

- **Node.js** 18+ — [скачать](https://nodejs.org)
- **FFmpeg** — установить одной командой:
  ```powershell
  winget install "FFmpeg (Essentials Build)"
  ```
- **Токен бота** lolka.app — [создать бота](https://lolka.app/developers/bots)

### Установка и запуск

1. Скачайте [релиз](https://github.com/AMURKA-PVE-SCUM/ym-bot-lolka/releases) или клонируйте репозиторий:
   ```bash
   git clone https://github.com/AMURKA-PVE-SCUM/ym-bot-lolka.git
   cd ym-bot-lolka
   ```

2. Запустите `start.bat` двойным кликом.

Скрипт автоматически:
- Установит все зависимости
- Запустит интерактивный визард настройки (при первом запуске)
- Проведёт авторизацию в Яндексе через устройство
- Сохранит токены в `.env`
- Запустит бота

> **Для продвинутых:** можно настроить вручную — скопируйте `.env.example` в `.env` и заполните токены.

## Команды

### Естественный язык (без префикса)

| Пример | Описание |
|--------|----------|
| `включи рок` | Запустить станцию по жанру |
| `поставь поп` | Запустить поп-музыку |
| `включи моя волна` | Персональное радио |
| `запусти чарт` | Топ Яндекс.Музыки |
| `включи Земфира — Искала` | Найти и воспроизвести трек |
| `включи новинки` | Свежие релизы |

### Текстовые команды

| Команда | Описание |
|---------|----------|
| `!ym <текст>` | Поиск треков |
| `!ym 1-5` | Выбрать трек из результатов поиска |
| `!ym wave` | Моя волна |
| `!ym radio` | Список станций |
| `!ym radio <номер>` | Запустить станцию |
| `!ym pl` | Список плейлистов |
| `!ym pl <номер>` | Воспроизвести плейлист |
| `!ym top` | Чарт |
| `!ym new` | Новинки |
| `!ym dislike` | Дизлайк текущего трека |
| `!skip` | Пропустить |
| `!stop` | Остановить и очистить очередь |
| `!prev` | Предыдущий трек |
| `!loop` | Цикл повтора: выкл → трек → очередь |
| `!shuffle` | Перемешать очередь |
| `!queue` | Показать очередь |
| `!np` | Что играет сейчас |
| `!help` | Справка |
| `!ym auth` | Авторизация в Яндексе (в чате) |
| `!ym retry` | Переподключить Яндекс.Музыку |

### Кнопки управления

```
⏮ ⏭ ⏹ 📋 🔀    — ряд 1: назад, вперёд, стоп, очередь, перемешать
🔁/🔂            — ряд 2: режим повтора (откл/трек/очередь)
◀ 📋 ▶            — при просмотре очереди: страницы
```

## Конфигурация `.env`

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| `LOLKA_TOKEN_YM` | ✅ | Токен бота lolka.app |
| `YM_TOKEN` | ✅ | Токен Яндекс.Музыки (заполняется setup.mjs) |
| `YM_SESSION_ID` | ❌ | Cookie для полных треков без Plus |
| `MUSIC_CHANNEL_ID` | ❌ | ID голосового канала для автоподключения |
